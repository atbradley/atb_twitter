atb_twitter

About

atb_twitter is a Textpattern plugin to display Twitter or StatusNet feeds on your website using Textpattern forms as display templates.

Developed by Adam T. Bradley.

Example

<txp:atb_twitter
   count="20"
   user="daydreamlab">
<ul class="tweets">
<li>

<span class="tweetdate">
   <txp:atb_tw_created_at />:
</span> 
<txp:atb_tw_text />

</li>

</txp:atb_twitter>
</ul>
